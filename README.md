## Run Locally

First of all Clone the project

```bash
  git clone https://github.com/bleedingAyush/messagely
```

Install dependencies

```bash
   npm run install-packages
```

Start the app

```bash
   npm run dev
```

Visit http://localhost:3000

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB
